== Description ==

**Custom Feed for TikTok is a WordPress plugin that lets you show your TikTok videos directly on your website, with a quick and simple setup.**

Connect your TikTok account, pick a layout, and embed your TikTok feed anywhere on your site using a shortcode. Your feed updates automatically every time you post on TikTok.

== Free Features ==

**Display Your TikTok Feed:** Connect your TikTok account and display your videos in a responsive grid gallery on any WordPress page or post.

**Grid Layout with Adjustable Columns:** Choose how many columns to display and set the spacing between videos. Works on all screen sizes.

**Control How Many Videos Show:** Set a specific number of TikTok videos to display. Useful for keeping pages clean and focused.

**Post Order:** Display videos in ascending or descending order based on when they were posted.

**Feed Settings:** The multiple feed setting options will allow you to open your post by default or on the TikTok site. Also, you can choose to display/hide the Author Name, Photo, TikTok video caption, or Platform Icon.

**Header Display:** Show your TikTok profile picture and account name above the feed, or hide the header entirely.

**Pagination:** Split your videos across multiple pages. Set how many videos appear per page and customize the pagination button text.

**Feed Caching:** Set a cache refresh interval to control how often the plugin fetches new data from TikTok. Improves performance and reduces API calls.

**Shortcode Embed:** Add your TikTok feed anywhere using the shortcode: `[wp_social_ninja id="template_id" platform="tiktok"]`

== How to Set Up Your TikTok Feed (Quick Start) ==

1. Install and activate **Custom Feed for TikTok** from the WordPress plugin directory
2. In your WordPress dashboard, go to **WP Social Ninja → Platforms → TikTok**
3. Generate your TikTok access code to connect your account
4. Click **Add New Template** and choose your layout
5. Configure your feed setting with number of videos, post order, header display, and more.
6. Click **Save** and copy the shortcode.
7. Paste the shortcode into any page, post, or widget. Your TikTok feed is live.

== Installation ==

This section describes how to install the plugin and get it working.
Install from WordPress Admin Panel:

1. Login to the WordPress Admin section
2. Go to Plugins > Add New
3. Type “Custom Feed for TikTok” on the Search bar and enter
4. Find this plugin and Click “Install Now”
5. Activate The Plugin

= Manual Installation =

1. Download the plugin from the WordPress.org repository
2. On your WordPress admin dashboard, go to Plugins > Add New > Upload Plugin
3. Upload the downloaded Plugin (custom-feed-for-tiktok.zip) file and click Install Now
4. Activate “Custom Feed for TikTok” from your Plugins page
5. Use Custom Feed for TikTok on your site from the WP Social Ninja dashboard

== Frequently Asked Questions ==

= What is a TikTok feed plugin for WordPress? =

A TikTok feed plugin for WordPress is a tool that connects your WordPress website to your TikTok account and automatically displays your TikTok videos in a customizable layout. Custom Feed for TikTok uses the TikTok Display API to fetch your videos and embed them on any page or post, without manual embedding or coding.

= How do I embed TikTok videos in WordPress? =

Install Custom Feed for TikTok, connect your TikTok account, create a feed template, and paste the shortcode `[wp_social_ninja id="template_id" platform="tiktok"]` into any page or post. Your TikTok feed will appear immediately and update automatically when you post new videos.

= Does this TikTok plugin work without coding? =

Yes. You can configure everything through a visual settings panel. No PHP or CSS needed.

= Can I display multiple TikTok feeds on one page? =

Yes. Create separate feed templates for different accounts or hashtags, then paste each shortcode into the page where you want it. You can display as many feeds as you need.

= Can I show TikTok feeds from multiple accounts? =

Yes, in the Pro version, you can connect multiple TikTok accounts and display them in separate feeds or combined into a single gallery.

= Does this plugin work with Elementor? =

Yes. Custom Feed for TikTok integrates natively with Elementor, Oxygen, and Beaver Builder. Add your TikTok feed widget to any page builder section without touching code.

= Does this plugin work after TikTok API changes? =

Yes. We regularly update the plugin to stay compatible with TikTok Display API. If your feed stops working, simply update to the latest version. If you still face issues, our support team is ready to help.

= Is this TikTok feed plugin GDPR compliant? =

Yes. WP Social Ninja is GDPR-compliant for social feeds. The plugin uses TikTok's official Display API and handles data according to TikTok's Terms of Service and Privacy Policy.

= What TikTok layouts are available? =

The free version includes a responsive Grid layout. The Pro version has Masonry and Carousel with autoplay, slide speed control, and navigation options.

= Where can I get support? =

[Submit a support ticket here](https://wpmanageninja.com/support-tickets/). Our team responds within 24 hours on business days.
